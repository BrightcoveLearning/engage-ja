    ビデオを管理する

    このトピックでは、アカウントにあるビデオを管理する方法を学習します。

    アカウントの動画にアクセスするには、にログインしてくださいBrightcoveEngage™をクリックしますビデオナビゲーションヘッダー内。

    ページの本体には以下が表示されます。

    • 動画サムネイルおよび長さ
    • 動画名
    • Video ID-システムによって割り当てられたビデオ ID（すべてのアカウントで一意です）
    • ステータス
      • -ビデオがアクティブであることを示します
      • -ビデオがアクティブでない（再生できない）ことを示します
    • 参照ID
    • 作成日時、ユーザーID
    • 更新された日付と時刻とユーザー ID

    動画のサムネイルの横にあるチェックボックスをクリックし、Shift キーを押しながら別のチェックボックスをクリックすると、複数の動画を選択できます。ビデオのリストは、列ヘッダーをクリックすることでソートできます。[動画] ページには「スティッキー」ソートが使用され、並べ替えの選択内容が記憶されます。移動して戻ったり、再度ログインしたりすると、[動画] ページには最後に選択した並べ替えが既定で表示されます。

    ページの上部にあるボタンバーには、次の機能があります。

    • アップロード -アカウントに動画をアップロードする
    • クイック編集 -ビデオプロパティのクイック編集
    • クイックビュー -ビデオのプロパティをすばやく表示する
    • もっと -に動画を追加するプレイリスト、アクティブ化/非アクティブ化ビデオとビデオを共有する他のアカウントへ
    • 削除 -アカウントから動画を削除します

    動画のプロパティ値を編集する

    動画プロパティを編集するには、動画名をクリックし、動画プロパティページを開きます。ビデオプロパティの編集については、「ビデオプロパティの編集」を参照してください。

    ザ・クイック編集この機能は、複数のビデオのプロパティを編集するときに役立ちます。

    動画をアップロードする

    アカウントへの動画のアップロードについては、を参照してください。動画のアップロード。

    クイック編集の使用

    クイック編集機能は、複数のビデオのプロパティを編集するときに役立ちます。詳細については、を参照してください。クイック編集を使用したビデオプロパティの編集。

    クイックビューを使用する

    クイックビューパネルを使用すると、ビデオのプロパティをすばやく表示できます。クイックビューパネルを開いたり閉じたりするには、[ クイックビュー ] をクリックします。パネルには、選択したビデオのビデオプロパティのサブセットが表示されます。

    クイックビューパネルは読み取り専用です。複数のビデオが選択されている場合、クイックビューには、最後に選択したビデオのプロパティが表示されます。

    再生リストへの動画の追加

    再生リストは、プレーヤーで再生するためにグループ化されたビデオのコレクションです。プレイリストを作成すると、BrightcoveEngageアプリケーションを作成するときに使用できます。プレイリストの作成と管理については、を参照してください。プレイリストの作成と管理。

    ビデオのアクティブ化と非アクティブ化

    ビデオページのステータス列は、ビデオが「アクティブ化」されているか、「非アクティブ化」されているかを示します。

    • -ビデオがアクティブであることを示します（ビデオは再生可能）
    • -ビデオが非アクティブであることを示します (ビデオを再生できません)

    ビデオを有効にするには、ビデオを選択し、その他アイコン [> アクティブ化 ] をクリックします。ビデオを無効にするには、ビデオを選択し、その他アイコン [> 非アクティブ化 ] をクリックします。[ ステータス ] 列が表示されない場合は、列セレクタを使用して選択します。

    動画を他のアカウントと共有する

    メディア共有によって、パブリッシャーは動画を一度アップロードすると、他のアカウントに動画を共有することができます。そのアカウントは、ご自身のアカウント、企業アカウント、他のパブリッシャーのアカウントのいずれでも構いません。動画のサムネイルには、動画がアカウントで共有されているか、別のアカウントで共有されているかが示されます。以下の例では、 CEOようこそビデオは別のアカウントに共有され、Brightcove ブの概要ビデオはこのアカウントに共有されました。

    他のアカウントとビデオを共有する方法の詳細については、を参照してください。共有ビデオの操作。

    動画を削除する

    動画を削除するには、削除する動画のサムネイルの横にあるチェックボックスをクリックし、削除ボタン（）をクリックします。[削除]ボタンをクリックして、削除を確認します。

    日付と時刻情報の作成と更新

    動画を作成または更新したユーザーのユーザー ID が [作成日] 列または [更新日] 列に表示されます。アクションを実行したユーザーがアカウントから削除された場合でも、その ID は引き続き表示されます。

    動画の変更によってユーザー ID が記録されない場合と、ユーザー ID が表示されない場合があります。

    • ビデオソースが置き換えられると、動画の更新日は更新されますが、アクションを開始したユーザーは記録されません
    • 画像（ポスター/サムネイル）および/またはテキストトラックが更新されると、動画の更新日は更新されますが、アクションを開始したユーザーは記録されません
    • 動画がアフィリエイトアカウントで共有されている場合、作成された日付に関連付けられたユーザー ID は表示されません。

    動画を検索する

    [検索] フィールドを使用すると、アカウント内の動画や再生リストを検索できます。詳細については、「動画を検索する方法」を参照してください。

    ビデオデータのエクスポート

    クリック書き出すすべてのビデオまたはビデオのサブセットのビデオメタデータを.csvファイルにエクスポートします。詳細については、を参照してください。 ビデオデータのエクスポート。

    列の表示をカスタマイズする

    ページの本文に表示される列は、列ボタン（）をクリックし、表示する列を選択することでカスタマイズできます。

    より多くの列が選択されると、ビデオリストを水平方向にスクロールしてすべての列を表示することができます。
