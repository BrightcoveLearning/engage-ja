    プレイリストを作成、管理する

    このトピックでは、プレイリストを作成および管理する方法について説明します。
     

    プレイリストは、ビデオプレーヤーで再生するために特定の順序でグループ化されるビデオのコレクションです。[ 動画 ] ページでは、プレイリストの作成や再生リストへの動画の追加が簡単に行えます。プレイリストを作成したら、Brightcove Engage™ アプリケーションで公開できます。

    プレイリストのタイプ

    プレイリストは、以下の2つのタイプがあります。

    • 手動 -手動プレイリストでは、特定の動画を再生リストに割り当てて、手動で並べ替えます。
    • スマート- スマートプレイリストでは、動画タグやその他の動画プロパティに基づいて、再生リストの動画の一覧が自動的に作成されます。スマートプレイリストは動的で、ビデオライブラリ内の動画とプレイリストを定義するルールに基づいて自動的に更新されます。

    プレイリストに追加できる動画は最大100本（手動とスマートの両方）。プレーヤーのプレイリストに最初に読み込まれる動画は、再生リストのタイプによって決まります。

    • 手動プレイリスト -動画は、手動再生リストのビデオの順序に基づいて再生リストに読み込まれます。
    • スマートプレイリスト - 再生順序プレイリストの設定に基づいて、ビデオがプレイリストに読み込まれます。

    プレイリストとモバイルアプリのエクスペリエンス

    モバイルアプリエクスペリエンスを作成する場合、エクスペリエンスの最初のビデオが注目動画として使用されます。注目のビデオがアプリケーションの上部に表示されます。

    ブライトコーブでは、表示される注目の動画を簡単に制御できるように、1 つの動画で注目動画プレイリストを作成することをお勧めします。これにより、プレイリストを編集して別の動画を選択することで、おすすめの動画を簡単に変更できます。これにより、エクスペリエンスを編集して再公開しなくても、おすすめの動画を簡単に変更できます。

    プレイリストを作成する

    再生リストを作成するには、[ 動画 ] ページを開き、次の手順に従います。

    1. サイドナビゲーションパネルで [プレイリストの作成] をクリックします。新規プレイリストダイアログが開きます。
      注：サイドナビゲーションボタン をクリックして、サイドナビゲーションパネルの表示と非表示を切り替えることができます。
    2. プレイリストのタイプを[手動]または[スマート]から選択します。
    3. 次のガイドラインに従って、プレイリスト名を入力します。
      • 名前は100文字に制限されています
      • 不正な文字:/
    4. （任意）[リファレンスID]および[説明]に入力します。
    5. サイドバーにプレイリストを表示するには、[サイドバーにプレイリストを表示]を選択します。
      注：サイドバーには最大 100 個のプレイリストが表示されます。
    6. [保存]をクリックします。

    スマートプレイリストを使用する

    スマートプレイリストを作成する場合は、以下の設定も使用できます。

    • タグ - タグを使用して、プレイリストに表示する動画を定義します。[ Tags ] ドロップダウンには、次のオプションがあります。
      • 1 つ以上のタグを含む -リストされている 1 つ以上のタグを含む動画を含める
      • すべてを含む -プレイリスト内のすべての動画にリストされているすべてのタグを持っていることを要求します

      [ Tags ] テキストフィールドに、プレイリストの定義に使用する 1 つまたは複数のタグ (カンマ区切り) を入力します。

    • 再生順序 - プレイリスト内の動画の再生順序を管理します。以下のオプションが利用できます。
      再生順序 説明
      アクティブ化された日（古い順） 動画がアクティブ化された日付に基づいて、最も古い動画が最初になります
      アクティブ化された日（新しい順） 動画が有効化された日付に基づいて、最新の動画が最初に表示されます
      ビデオ名 (A-Z) 動画名のアルファベット順
      総再生回数順 合計ビデオ再生数に基づきます
      先週よく再生された順 最後の週の総動画再生数に基づきます
      開始日（古い順） 動画の開始予定日に基づいて、最も古い動画が最初になります。開始日が指定されていない動画は、開始日で並べ替えられるプレイリストには表示されません。
      開始日（新しい順） 動画の開始予定日に基づいて、最新の動画が最初に表示されます。開始日が指定されていない動画は、開始日で並べ替えられるプレイリストには表示されません。
    • 動画数の制限 - プレイリストに入れる動画の最大数を入力します。

    手動プレイリストに動画を追加する

    動画を手動プレイリストに追加するには、次の手順を行います。

    1. プレイリストに追加する動画の隣のボックスを選択します。
    2. ページ上部の [ その他] [> プレイリストに追加 ] をクリックします。
    3. 動画を追加するプレイリストを選択します。
    4. [追加]をクリックします。手動プレイリストに追加された動画は、プレイリストの一番上に追加されます。

    プレイリスト内の動画の順序を変更する

    プレイリストの動画の順序を変更するには、使用しているプレイリストのタイプに応じた手順を行います。

    手動再生リストのビデオの順序を変更する

    1. 以下の方法の1つを使用して、プレイリストを開きます。
      • サイド・ナビゲーション・パネルのプレイリスト名をクリックします
      • サイド・ナビゲーション・パネルのすべてのプレイリストをクリックし、プレイリストを見つけて、プレイリスト名のリンクをクリックします。
    2. プレイリスト内の動画のリストが表示されます。
    3. 動画をクリックし、順序の新しい場所にドラッグします。
    注：再生順フィールドを編集して、ビデオの順序を変更することもできます。

    スマートプレイリストの動画の順序を変更する

    1. そのプレイリストがサイドバーに表示されている場合、プレイリストをクリックし、次にプレイリストの隣の下向き矢印をクリックし、[設定の編集]をクリックします。
    2. そのプレイリストがサイドバーに表示されていない場合、サイドナビゲーションの[すべてのプレイリスト]見出しをクリックします。次にプレイリストを見つけて、プレイリスト名リンクをクリックし、[設定の編集]をクリックします。
    3. [再生順序]で、そのプレイリストの新しい再生順を選択します。
    4. [保存]をクリックします。

    スマートプレイリストは、プレーヤーが利用可能なビデオ情報から読み込まれたときに動的に作成されるため、注文は常に現在の状態を反映する必要があります。

    プレイリストの設定を編集する

    手動プレイリストまたはスマートプレイリストの設定を編集するには、次の手順に従います。

    1. そのプレイリストがサイドバーに表示されている場合、プレイリストをクリックし、次にプレイリストの隣の下向き矢印をクリックし、[設定の編集]をクリックします。
    2. そのプレイリストがサイドバーに表示されていない場合、サイドナビゲーションの[すべてのプレイリスト]見出しをクリックします。次にプレイリストを見つけて、プレイリスト名リンクをクリックし、[設定の編集]をクリックします。
    3. プレイリストの設定を編集し、[保存]をクリックします。

    そのプレイリストがサイドナビゲーションパネルに表示されている場合、プレイリストをクリックし、次にプレイリストの隣の下向き矢印をクリックし、[設定の編集]をクリックします。

    プレイリストを削除する

    プレイリストを削除するには、次の手順を行います。

    1. すべてのプレイリストのリストを表示するには、[すべてのプレイリスト]をクリックします。
    2. 削除するプレイリストの隣のチェックボックスをチェックします。
    3. 削除ボタン（）をクリックします。
    4. [ プレイリストを削除 ] をクリックして、削除を確定します。

    そのプレイリストがサイドナビゲーションパネルに表示されている場合、プレイリストをクリックし、次にプレイリストの隣の下向き矢印をクリックし、[削除]をクリックします。

    手動プレイリストから動画を削除する

    手動再生リストから動画を削除するには、次の手順に従います。

    1. 次のいずれかの方法を使用して、手動プレイリストを開きます。
      • サイド・ナビゲーション・パネルのプレイリスト名をクリックします
      • サイド・ナビゲーション・パネルのすべてのプレイリストをクリックし、プレイリストを見つけてプレイリスト名をクリックします。
    2. プレイリスト内の動画のリストが表示されます。
    3. 削除する動画の隣のチェックボックスをチェックします。
    4. [ ビデオを削除] [> プレイリストから削除 ] をクリックします。

    また、動画プロパティを編集して、手動プレイリストから動画を削除することもできます。[ ビデオ情報 ] セクションで、[ 手動プレイリスト] セクションから再生リスト名を削除します。
