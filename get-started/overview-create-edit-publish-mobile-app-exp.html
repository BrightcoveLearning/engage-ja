概要:モバイルアプリエクスペリエンスの作成、編集、公開

このトピックでは、モバイルアプリエクスペリエンスを作成、編集、公開するための基本的な手順について説明します。

ブライトコーブ・エンゲージ（Brightcove Engage）™では、技術的なリソースを必要とせずに、魅力的な動画体験を作成できます。以下は、Brightcove Engage モバイルアプリテンプレートを使用して構築されたモバイルアプリエクスペリエンスの例です。

モバイルアプリエクスペリエンスを構築するためのステップバイステップガイドについては、ステップバイステップをご覧ください。モバイルアプリエクスペリエンスの作成と公開。

モバイルアプリエクスペリエンスを作成および公開する主な手順は次のとおりです。

  • テンプレートを選択
  • 動画を追加してエクスペリエンスをカスタマイズする
  • エクスペリエンスをプレビューして公開する

テンプレートの選択

新しいエクスペリエンスを作成するときは、テンプレートを選択することから始めます。テンプレートは、エクスペリエンスの全体的なルックアンドフィールを制御します。テンプレートの詳細およびサンプルサイトについては、「エクスペリエンステンプレートの概要」トピックを参照してください。

テンプレートを選択した後、サンプルコンテンツをエクスペリエンスに事前入力するオプションがあります。[ はい、エクスペリエンスを事前入力する] をクリックすると、Brightcove アカウントの動画がエクスペリエンスに事前入力されます。これにより、カスタマイズプロセスを開始するにつれて、エクスペリエンスの外観を簡単に確認できます。

動画の追加とエクスペリエンスのカスタマイズ

エクスペリエンスの作成後、エクスペリエンスエディターを使用して、エクスペリエンスのさまざまな側面をカスタマイズします。次の種類のカスタマイズを実行できます。

  • ページ -必要に応じてロゴをホームページに追加して、エクスペリエンスをブランド化することができます。ロゴの追加の詳細については、「モバイルアプリエクスペリエンスへのロゴの追加」を参照してください。
  • ビデオ -コレクションは、ビデオコンテンツを選択して整理するために作成されます。エクスペリエンスにビデオコンテンツを追加する方法の詳細については、「エクスペリエンスへのビデオの追加」を参照してください。
  • スタイル -新しいテーマを選択するか、サイトの色を変更することで、エクスペリエンスの外観をカスタマイズできます。スタイルのカスタマイズの詳細については、「モバイルアプリエクスペリエンスのスタイル設定」を参照してください。
  • アクセス制御 -シングルサインオンを使用して公開アプリへのアクセスを制限できます。アクセス制御の詳細については、「エクスペリエンスへのアクセスの制御」を参照してください。
  • 概要 -エクスペリエンスの名前、説明、言語を編集できます。サイトの概要の編集の詳細については、「エクスペリエンスの詳細を編集する」を参照してください。

エクスペリエンスのプレビューと公開

モバイルアプリエクスペリエンスをプレビューすると、エクスペリエンスがモバイルデバイス上でどのように表示されるかを確認できます。 エクスペリエンスを新しいブラウザータブでプレビューしたり、共有可能なリンクを生成して、他のユーザーがサイトをプレビューすることができます。エクスペリエンスを公開すると、 Brightcove Engage モバイルアプリに必要なアクセスキーが生成されます。エクスペリエンスのプレビューと公開の詳細については、「モバイルアプリエクスペリエンスのプレビューと公開」を参照してください。
