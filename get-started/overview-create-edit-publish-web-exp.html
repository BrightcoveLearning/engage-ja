概要:Web エクスペリエンスの作成、編集、公開

このトピックでは、Web エクスペリエンスを作成、編集、公開するための基本的な手順について説明します。

ブライトコーブ・エンゲージ（Brightcove Engage）™では、技術的なリソースを必要とせずに、魅力的な動画体験を作成できます。ウェブエクスペリエンスはレスポンシブデザインの原則に従っているため、デスクトップ、タブレット、モバイルデバイスでも見栄えがよくなります。

以下は、Brightcove Engage ウェブテンプレートを使用して構築された Web エクスペリエンスの例です。

Web エクスペリエンスの構築に関するステップバイステップガイドについては、ステップバイステップをご覧ください。Web エクスペリエンスの作成と公開

Web エクスペリエンスを作成および公開する主な手順は次のとおりです。

  • テンプレートを選択してください
  • 動画を追加してエクスペリエンスをカスタマイズする
  • エクスペリエンスをプレビューして公開する

テンプレートの選択

新しいエクスペリエンスを作成するときは、テンプレートを選択することから始めます。テンプレートは、エクスペリエンスの全体的なルックアンドフィールを制御します。Brightcove Engage には一連のテンプレートが用意されており、視聴者とエンゲージメントするために必要な動画体験の種類を柔軟に配信できます。テンプレートの詳細およびサンプルサイトの表示については、「エクスペリエンステンプレートの概要」トピックを参照してください。

テンプレートを選択した後、エクスペリエンスにサンプルコンテンツを事前入力するオプションがあります。[ はい、エクスペリエンスを事前入力する] をクリックすると、アカウントの動画がエクスペリエンスに事前入力されます。これにより、カスタマイズプロセスを開始するにつれて、エクスペリエンスの外観を簡単に確認できます。

動画の追加とエクスペリエンスのカスタマイズ

エクスペリエンスの作成後は、エクスペリエンスエディターを使用してエクスペリエンスをカスタマイズします。次のタイプのカスタマイズを実行できます。

  • ページ -必要に応じて、画像、テキスト、リンクをページテンプレートに追加して、エクスペリエンスをブランド化することができます。ページへのオブジェクトの追加の詳細については、「Web エクスペリエンスページのコンテンツのカスタマイズ」を参照してください。
  • ビデオ -コレクションは、ビデオコンテンツを選択して整理するために作成されます。サイトへの動画コンテンツの追加の詳細については、「エクスペリエンスへのビデオの追加」を参照してください。
  • スタイル -新しいテーマを選択するか、サイトの色やフォントを変更することで、エクスペリエンスの外観をカスタマイズできます。スタイルのカスタマイズの詳細については、「Web エクスペリエンスのスタイルのカスタマイズ」を参照してください。
  • アクセス制御 -サイトへのアクセスを制限します。サイト構成の詳細については、「エクスペリエンスへのアクセスを制御する」を参照してください。
  • 概要 -エクスペリエンスの名前、説明、言語を編集します。サイトの概要の編集の詳細については、「エクスペリエンスの詳細を編集する」を参照してください。

エクスペリエンスのプレビューと公開

エクスペリエンスをプレビューすると、モバイル、タブレット、デスクトップデバイスでエクスペリエンスがどのように表示されるかを確認できます。 エクスペリエンスを新しいブラウザータブでプレビューしたり、共有可能なリンクを生成して、他のユーザーがサイトをプレビューすることができます。エクスペリエンスを公開すると、サイトの静的 URL が生成され、閲覧者がアクセスできるようになります。Web エクスペリエンスのプレビューと公開の詳細については、「Web エクスペリエンスのプレビューと公開」を参照してください。
