エクスペリエンスエディターを使用した Web エクスペリエンスのカスタマイズ

このトピックでは、エクスペリエンスエディターを使用して Web エクスペリエンスのルックアンドフィールをカスタマイズする方法について説明します。

エクスペリエンスエディターは、Brightcove Engage (Brightcove Engage) を使用して構築されたエクスペリエンスのルックアンドフィールをカスタマイズするために使用されます™。エクスペリエンスのタイトルをクリックして、エクスペリエンスエディターを開きます。Web エクスペリエンスでは、エクスペリエンスエディターには次の機能があります。

  • ページへのオブジェクトの追加
  • サイトに動画を追加する
  • サイトスタイルのカスタマイズ
  • サイトへのアクセスを制御する
  • サイトの詳細の構成

これらのオプションにアクセスするには、左側のナビゲーションのメニューをクリックします。

ページにオブジェクトを追加する

テキスト、画像、URLなどのオブジェクトをページに追加して、Webエクスペリエンスをカスタマイズしてブランド化することができます。ページにオブジェクトを追加する方法については、「Web エクスペリエンスページへの画像、テキスト、リンクの追加」を参照してください。

Web エクスペリエンスへのビデオの追加

エクスペリエンスにビデオを追加するには、をクリックしますビデオ左側のナビゲーションで。Web エクスペリエンスにビデオを追加する方法の詳細については、「エクスペリエンスへのビデオの追加」を参照してください。

スタイルのカスタマイズ

Webエクスペリエンスで使用される色とフォントをカスタマイズする、または別のテーマを選択するには、をクリックします。スタイル左側のナビゲーションで。スタイルをカスタマイズする方法の詳細については、「Web エクスペリエンスのスタイルのカスタマイズ」を参照してください。

サイトへのアクセスを制御する

アクセス制御設定は、サイトへのアクセスを制限するために使用されます。アクセス制御設定を構成するには、左側のナビゲーションにある [ アクセス制御 ] をクリックします。

アクセス制御設定の詳細については、「エクスペリエンスへのアクセスを制御する」を参照してください。

エクスペリエンスの詳細の設定

[概要] 設定では、エクスペリエンスの名前、説明、言語を編集できます。エクスペリエンスの詳細を構成するには、をクリックします概要左側のナビゲーションで。

サイト詳細の構成の詳細については、「エクスペリエンスの詳細の編集」を参照してください。
