Auth0 パスワードレス認証を使用したエクスペリエンスの保護

このトピックでは、Auth0 パスワードレス認証を使用してエクスペリエンスを保護する方法について説明します。

エクスペリエンスは、パスワードレス認証で Auth0 を使用してセキュリティで保護できます。このセキュリティ方法を使用する場合、ユーザーは電話番号の入力を求められます。Auth0 は Twilio を使用して 1 回限りのコードを送信します。その後、ユーザーはアプリケーションにコードを入力します。コードに添付されている電話番号が既存のユーザと一致する場合、Auth0 はユーザーを認証します。

注:このトピックの手順と画像は、 Auth0と Twilioの試用版を使用しています。

始める前に

Auth0 パスワードレスを設定する前に、次のタスクを完了します。

  • 「Auth0 を使用したエクスペリエンスのセキュリティ保護」で説明されているように、Auth0 を使用してモバイルアプリエクスペリエンスを構成する
  • Twilio トライアルアカウントにサインアップし、アカウントを確認する

Twilio の電話番号を取得する

パスワードレスを設定するときに、送信者番号として使用される Twilio 電話番号を取得する必要があります。

  1. Twilio アカウントにログインします。
  2. [ ダッシュボード] ページで、[ 試用番号の取得] をクリックします。
  3. [ この番号を選択 ] をクリックします。
  4. [ 完了 ] をクリックして番号を確認します。

Auth0 でのパスワードレス接続の構成

まず、Auth0 でパスワードレス接続を設定します。

  1. Auth0 アカウントにログインします。
  2. 左側のナビゲーションで、[ 接続] > [パスワードなし] の順にクリックします。
  3. SMS 接続を有効にします。

Twilio SIDを入力する

  1. Twilio に戻ります。
  2. [ダッシュボード] ページで、[ アカウント SID ] フィールドを見つけます。
  3. 値をクリップボードにコピーします。
  4. Auth0 に戻ります。
  5. 値を Twilio SID フィールドに貼り付けます。

Twilio 認証トークンを入力する

  1. トゥイリオに戻れよ
  2. [ダッシュボード] ページで、[ 認証トークン ] フィールドを見つけます。
  3. [ Show] リンクをクリックして、トークンを表示します。
  4. 値をクリップボードにコピーします。
  5. Auth0 に戻ります。
  6. [ Twilio 認証トークン ] フィールドに値を貼り付けます。

Twilio From 電話番号を入力する

  1. トゥイリオに戻れよ
  2. [ダッシュボード] ページで、前に作成したトライアル番号を見つけます。
  3. 番号をクリップボードにコピーします。
  4. Auth0 に戻ります。
  5. [ From] フィールドに値を貼り付けます。
  6. （オプション）メッセージテキストをカスタマイズします。

アプリケーションの有効化

  1. ダイアログの上部にある [ アプリケーション] リンクをクリックします。
  2. 作成した Auth0 アプリケーションを有効にして、Brightcove Engage アプリケーションを保護します。
  3. [ 保存 ] をクリックしてダイアログボックスを閉じます。

パスワードレスログインを使用するようにアプリケーションを設定する

  1. 左側のナビゲーションで、[ ユニバーサルログイン] をクリックします。
  2. 「ログイン」リンクをクリックします。
  3. [ ログインページのカスタマイズ ] オプションを有効にします。
  4. [ 既定のテンプレート ] ドロップダウンリストから [ ロック (パスワードなし) ] を選択します。
  5. [ 変更を保存 ] をクリックします。

エクスペリエンスのプレビュー

上記の手順が完了したら、体験をプレビューして、視聴者に電話番号の入力を求められることを確認できます。

  1. ブライトコーブ Engage にログインします。
  2. Auth0 で保護されているエクスペリエンスのタイトルをクリックします。
  3. エクスペリエンスをプレビューします。電話番号の入力を求めるログイン画面が表示されます。
  4. プレビューを閉じます。

 

Twilioに有効な電話番号を追加する

テストのために、Twilioに1つ以上の電話番号を追加します。

  1. トゥイリオに戻れよ
  2. 左側のナビゲーションで、[ すべての製品とサービス] アイコンをクリックします。
  3. 左側のナビゲーションで、[ 電話番号] をクリックします。
  4. 左側のナビゲーションで、[ 確認済みの着信側 ID] をクリックします。
  5. テストに使用する数値を入力します。トライアルアカウントには、トライアルアカウントを作成したときに使用した電話番号が必要です。

構成の検証

最後のステップは、視聴者が自分の携帯電話番号を使用してログインできるかどうかをテストすることです。

  1. デバイスで Brightcove Engage アプリケーションを開きます。プロンプトが表示されたら、エクスペリエンスのアクセスコードを入力します。
  2. 電話番号を入力し、[ 送信 ] をクリックします。
  3. 確認コードが記載された SMS メッセージを受信することを確認します。
  4. 確認コードを入力し、[ 送信 ] をクリックします。

Brightcove Engage モバイルアプリが表示されます。
